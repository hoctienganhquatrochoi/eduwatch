## 📋 Thêm video mới
1. Vào Admin → tab Video → + Thêm video
2. Dán link YouTube → nhấn Tự động lấy transcript
3. Nhấn Lưu

## 👨‍🎓 Tạo học sinh mới
1. Vào Admin → tab Học sinh → + Thêm học sinh
2. Điền Họ tên, Username, Mật khẩu
3. Nhấn Lưu → gửi username + password cho học sinh
